Le site web présente la formation du BUT et plus particulièrement celle du BUT informatique au sein de l'IUT Lyon 1 - Villeurbane la Doua. A l'intérieur de ce dernier vous pourrez trouver des informations sur la formation, le programme, l'univeristé mais également des orientations possibles après la formation. Enfin une page de contact est également disponible pour obtenir plus de renseignements.
